# People's Patterns — Upsell / Cross-Sell / Downsell Strategy

## Upsells (increase order value)

### A0 / Copy Shop File — +$4
**What:** In addition to the tiled Letter/A4 PDF, offer a single-sheet A0 or plotter-size file. User takes it to a copy shop (Staples, FedEx) and prints the entire pattern on one sheet — no taping.
**When to offer:** On the checkout page as an add-on checkbox.


### Projector File — +$4
**What:** A single-page PDF or SVG optimized for projector sewing — no seam allowances baked in (projector sewists add their own), clean lines, high contrast, calibration marks.
**When to offer:** Same checkout page add-on.
**Note:** Build this when there's demand — projector sewing is growing fast but is still a smaller segment.

### Fit Insurance — +$6
**What:** After sewing, if the fit is off, user submits feedback and gets one free re-draft with adjusted measurements. Normally re-generation is free for measurement updates, but Fit Insurance includes a personal fit consultation via email — you review their feedback and suggest specific measurement adjustments.
**When to offer:** Checkout page, framed as "peace of mind for your first pattern."

### Sew-Along Video Access — +$9
**What:** A detailed video walkthrough for that specific garment, showing every step from cutting to finishing.
**When to offer:** On the pattern page and at checkout.
**Note:** Only offer this once you have the video filmed. Start with cargo shorts and tee.

### Capsule Add-On — 25% off
**What:** "Add the matching [skirt/short/blouse] for 25% off." Suggests a complementary garment from the same or related body block.
**When to offer:** Post-purchase confirmation page and recommendation email.

## Cross-Sells (related products at point of purchase)

### Same Block Patterns
- "This pattern uses the same body block as [X], [Y], and [Z]. Your measurements are already ready."
- Show on every pattern page and post-purchase page
- These are the highest-converting cross-sells because zero additional measurement effort

### Fabric + Notions Affiliate Links
- Every pattern's materials list recommends specific fabrics and notions
- Link each recommendation to an affiliate partner
- "Cotton twill 8-10oz" links to Mood Fabrics or Amazon search
- "Universal needle 80/12" links to Amazon
- Passive revenue on every pattern generated



### Starter Kit
- Physical product: branded tape measure, chalk, pins, seam ripper, thread snips
- Offer at checkout: "Get everything you need to start — $29"
- Ship via Amazon FBA or direct fulfillment
- Only build this when you have demand signal (email signups asking for it)

### Bundle Deals
- 3-pattern capsule: $29 (saves $2-12 depending on pattern tiers)
- 5-pattern wardrobe: $49 (saves $6-30)
- Show on pricing page and in checkout as "save with bundles"

## Downsells (prevent losing the customer)

### Free First Pattern
- New accounts get 1 free download
- No credit card required
- Removes all purchase anxiety
- They experience the full product before paying

### Save Profile Without Buying
- If someone generates but doesn't buy, offer: "Save your measurement profile for next time — free"
- Gets them to create an account without payment
- You capture the email for nurture flows
- When they come back, measurements are pre-filled

### Exit-Intent Offer
- When mouse moves to close tab (desktop), show a subtle banner
- "Before you go — save your measurements and get your first pattern free"
- Not aggressive, not a popup — a slide-in bar



### Discount Code for Hesitant Buyers
- FIRST50 — 50% off first pattern
- Share in the abandon cart email
- Share on Reddit launch post
- Time-limited to create urgency

### Re-Download Assurance
- "You can re-download anytime"
- "Update measurements and re-generate for free"
- "30-day money-back guarantee"
- These aren't discounts but they reduce the perceived risk of buying

## When to Show Each Offer

| Moment | Offer |
|--------|-------|
| Pattern page (browsing) | Related patterns from same block, bundle pricing |
| Checkout page | A0 file, projector file, fit insurance, sew-along video |
| Post-purchase confirmation | Capsule add-on at 25% off, "these patterns use your measurements too" |
| Account dashboard | Wardrobe progress, next recommended pattern, bundle upgrade |
| Email: generated not purchased | Free first pattern reminder, FIRST50 code |
| Email: cart abandon | FIRST50 code, "your pattern is still ready" |
| Email: 21 days post-purchase | Next pattern recommendation, capsule bundle |
| Exit intent | Save profile free, first pattern free |

## Revenue Model Per Customer Journey

**Minimum journey:** Free pattern → nothing else = $0 (but you got the email)

**Typical journey:** Free pattern → 1 paid pattern ($9-14) → 1 more in 60 days ($9-14) = $18-28 in 6 months

**Power user journey:** Free pattern → 3 patterns ($30-42) → membership ($12/mo) → annual ($120-240) → A0 upsells → affiliate clicks = $200-400+ year 1

**Best customer:** Membership + regular purchases + fit feedback + referrals + content sharing = $300+/year + brings in 2-3 new customers
