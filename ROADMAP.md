# Roadmap

## Phase 1 — MVP Launch

### App
- [ ] Wizard UI — 4-step flow: Choose → Measure → Customize → Pattern (full-screen each step, stepper bar, slide/fade transitions)
- [ ] React + Tailwind migration
- [ ] Pattern output tabs: Pieces · Materials · Instructions · Print
- [ ] Print-ready PDF — tiled (home printer) + full-size (A0/plotter), scale verification square, print instructions on page
- [ ] Seam allowance toggles on print output

### Patterns — 6 to ship at launch
- [ ] Cargo Shorts — muslin tested, geometry verified
- [ ] Straight Leg Jeans — muslin tested
- [ ] T-Shirt — muslin tested
- [ ] Button-Up / Camp Shirt — muslin tested
- [ ] A-Line Skirt — muslin tested
- [ ] Wide-Leg Trouser — muslin tested

### Marketing
- [ ] Domain
- [ ] Instagram handle — 2–3 weeks of content before launch
- [ ] TikTok handle — 2–3 weeks of content before launch
- [ ] YouTube handle
- [ ] Content types: build-in-public, behind the scenes, sewing the patterns, tiny tutorials
- [ ] Reddit launch — r/sewing, r/sewhelp, r/learnmachinesewing
- [ ] Facebook sewing groups — "free custom fit patterns, just enter your measurements"

### Landing page
- [ ] 10-second explanation above the fold
- [ ] Email capture for free tier
- [ ] "How to measure yourself" video embed
- [ ] Sew-along video for best launch pattern

## Phase 2 — Monetization

### Payments
- [ ] Stripe checkout — per-pattern purchase ($9–15)
- [ ] Monthly membership — unlimited or throttled downloads, saved profiles, fit history, pattern library access, discounts
- [ ] Gift cards (Stripe)
- [ ] Cart flow for multi-pattern checkout

### Accounts
- [ ] Account + login required to generate/download
- [ ] Saved measurement profiles tied to account
- [ ] Re-download with different measurements (named saves: "Me Jan 2026", "Me Mar 2026")
- [ ] Pattern version history
- [ ] Fit history log

### Pattern protection
- [ ] Watermarked low-res preview until purchase
- [ ] Full print-ready file unlocked after payment
- [ ] Downloaded patterns stamped with: customer name, order number, email, date
- [ ] Rate-limit downloads
- [ ] Commercial-use terms for microbrands and businesses
- [ ] Patterns stored in account history, not public links

### Upsells
- [ ] Branded beginner sewing tool kit ($25–35) — upsell at first pattern purchase
- [ ] Affiliate links — fabric companies, Amazon tools

---

## Phase 3 — Growth

### Referrals & retention
- [ ] Referral program — "Give a friend a free pattern, get a free pattern"
- [ ] Seasonal collections — quarterly themed drops (e.g. "Summer Linen Collection": easy pants, camp shirt, A-line skirt) with email + content campaign

### Fit improvement
- [ ] Fit feedback loop — post-sew prompt ("how did it fit?") with structured options (too tight at chest, crotch too low, etc.) to improve generated geometry over time

### Advanced customization
- [ ] User-adjustable pleats, pockets, rise, silhouette, cuff, waistband width, leg width — in-UI with live pattern update

### Output formats
- [ ] Layered PDFs — A4, Letter, A0, plotter with per-layer seam allowance toggles and fabric cut plan

### Content & education
- [ ] Per-garment YouTube sew-along: measure → generate → print → cut → sew → finished
- [ ] "How to measure yourself correctly" video series
- [ ] Sewing tape measure as branded product

### Audience expansion
- [ ] Fashion students and schools (bulk / institutional pricing)
- [ ] Indie designers and made-to-order microbrands (commercial license tier)
- [ ] Cosplay / costume community

- Women sick of standard sizing
- Indie fashion designers and made-to-order microbrands
