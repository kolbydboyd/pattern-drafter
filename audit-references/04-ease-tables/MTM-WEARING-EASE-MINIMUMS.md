# MTM Wearing Ease Minimums

Source: Industry consensus; RTW wearing-ease floors reduced for MTM because
there is no sizing-tolerance buffer (RTW patterns add extra ease to accommodate a
range of bodies within a size category; MTM is already sized to the individual).

---

## Wearing vs Design Ease

- **Wearing ease**: minimum extra fabric so the wearer can move, sit, breathe.
  Cannot be reduced below these minimums regardless of fit preference.
- **Design ease**: additional fullness for silhouette. Can be zero or negative
  (for stretch fabrics).

---

## MTM Wearing Ease Minimums

| Body part | Minimum ease |
|-----------|-------------|
| Bust / chest | ≥ 1" (2.5 cm) |
| Waist | ≥ ¾" (2 cm) |
| Hip / seat | ≥ 1" (2.5 cm) |
| Biceps | ≥ ¾" (2 cm) |
| Thigh | ≥ 1" (2.5 cm) |
| Neck | ≥ ½" (1.3 cm) |

---

## Difference from RTW

MTM typically uses less design ease and zero sizing-tolerance buffer.
RTW patterns must accommodate a range of bodies within a size, so they add 1–2"
extra beyond wearing ease. MTM does not need this buffer.

Therefore, an MTM "fitted" option can use 1–2" total ease (at or near wearing-ease
minimum) where RTW "fitted" would need 3–4".

Note: "slim" lower-body ease (+2.5") is labeled as only appropriate for stretch
fabrics in the engine comment (`geometry.js:509`). For woven fabrics, the hip ease
minimum is technically met (2.5 > 1") but may feel uncomfortably snug.
